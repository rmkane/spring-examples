üéØ How Desync Works

Each task uses the @Desync annotation with:

  • Stable per-instance splay based on key + appName + host
  • Per-run random component to reduce collisions
  • Per-run jitter to prevent long-term re-alignment

This prevents multiple instances from executing the same scheduled task simultaneously.
